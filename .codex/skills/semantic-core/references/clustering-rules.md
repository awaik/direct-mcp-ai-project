# Clustering Rules

Кластеризация нужна для управляемой рекламы, а не для красивой таблицы.

## Базовый Принцип

Один кластер должен иметь:

- единый интент;
- одну релевантную посадочную;
- похожую температуру спроса;
- похожую экономику;
- достаточно объёма для обучения стратегии;
- объявления, которые естественно покрывают все фразы группы.

## Под Автостратегии

Не делать SKAG по умолчанию. Для автостратегий лучше укрупнять группы, если:

- фразы ведут на одну посадочную;
- объявления остаются релевантными;
- CPA/ДРР ожидаемо близкие;
- нет сильной разницы в температуре.

Разделяй группы, если:

- нужны разные посадочные;
- разные ставки/ценность лида;
- разные регионы или доставка;
- разные УТП;
- смешаны бренд, конкуренты и generic;
- в одной группе появляются разные интенты.

## Кампании

Обычно разделяй по:

- поиск / РСЯ;
- гео;
- бренд / конкуренты / generic;
- температура спроса;
- маржинальность или продуктовая категория;
- разные цели или посадочные;
- отдельные бюджеты и стратегии.

## SERP-Clustering

Если по двум запросам Яндекс показывает примерно одни и те же типы сайтов и посадочных, это аргумент за один кластер. Если выдача разная, разделяй даже при похожих словах.

## Кросс-Минусация

Делай после финальной структуры:

- между конкурирующими кластерами;
- между брендовой и generic-кампанией;
- между категориями с пересекающимися словами;
- между горячими и информационными кластерами.

Не добавляй кросс-минусы, если они могут отрезать ценный длинный хвост.
