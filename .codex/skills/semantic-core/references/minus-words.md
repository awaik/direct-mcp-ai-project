# Minus Words

Минус-слова должны снижать мусорный расход, а не слепо обрезать спрос.

## Уровни

| Уровень | Что минусовать |
|---|---|
| Аккаунт | общий мусор, который почти никогда не нужен |
| Кампания | нерелевантные категории, гео, тип спроса |
| Группа | пересечения с соседними группами и уточнения интента |

## Частые Общие Минусы

Использовать только после проверки ниши:

- бесплатно, скачать, торрент, реферат, курсовая;
- своими руками, самостоятельно, инструкция, форум;
- вакансия, работа, зарплата, обучение, курсы;
- б/у, авито, юла, бу;
- фото, картинки, видео;
- отзывы, если это не отдельная тёплая стратегия;
- дешево/недорого, если бизнес не работает с эконом-сегментом.

## Нельзя Автоматически Минусовать

- "отзывы" — может быть коммерческим сравнением.
- "цена" и "стоимость" — обычно горячий спрос.
- "рядом", "адрес", "телефон" — полезно для локального бизнеса.
- "как выбрать" — может быть полезно для SEO и верхней воронки.
- бренды — только по правилам брендовой/конкурентной стратегии.

## Формат Рекомендации

Для каждого минуса указывай:

- уровень: аккаунт / кампания / группа;
- слово или фраза;
- причина;
- риск потери спроса: низкий / средний / высокий.

## После Запуска

Проверять отчёт по поисковым запросам:

- крупные аккаунты: ежедневно в первые дни;
- обычные кампании: минимум раз в неделю;
- после стабилизации: по объёму расхода и CPA.
